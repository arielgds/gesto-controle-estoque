# ChurrasControl 🥩🔥

O **ChurrasControl** é um aplicativo móvel desenvolvido com Flutter projetado para gerenciar o estoque de insumos para churrasco. Seja para uso pessoal ou pequenos eventos, ele ajuda a garantir que nunca falte carne ou carvão na hora da brasa.

## 🚀 Funcionalidades

- **Dashboard de Churrasco**: Acompanhe o total de itens e alertas de estoque crítico (abaixo de 2kg ou 2 unidades).
- **Gestão por Peso e Unidade**: Suporte para Kg, g, Unidades e Packs.
- **Categorias Específicas**: Organize por Carnes, Bebidas, Acompanhamentos, Carvão e mais.
- **Ações Rápidas**: Adicione ou remova quantidades (em passos de 0.5) com um toque.
- **Nuvem com Firebase**: Dados sincronizados em tempo real no Firestore.

## 🛠️ Tecnologias Utilizadas

- [Flutter](https://flutter.dev/) - Framework UI.
- [Firebase Firestore](https://firebase.google.com/) - Banco de dados NoSQL na nuvem.
- **Material 3** - Design moderno com cores temáticas.

## 🏁 Como Começar

### Pré-requisitos
- Flutter SDK instalado.
- Projeto configurado no Firebase Console.

### Instalação

1. Clone este repositório.
2. Configure o Firebase rodando `flutterfire configure`.
3. Execute `flutter pub get`.
4. Rode o app com `flutter run`.
